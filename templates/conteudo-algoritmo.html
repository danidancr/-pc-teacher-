Exercícios de Fixação

1. Qual a característica essencial de um algoritmo que o torna replicável e previsível?

Ser simples o suficiente para ser memorizado por qualquer pessoa.
Ter um número finito de passos e instruções não-ambíguas.
Incluir sempre, no mínimo, três estruturas de decisão (SE/ENTÃO/SENÃO).
Ignorar todos os detalhes complexos do problema original.

2. Qual componente é essencial para um algoritmo pedagógico quando uma ação precisa ser repetida até que uma condição seja satisfeita?

A Decomposição, para que o problema não seja muito longo.
Estrutura de Repetição (ou "Loop").
Estrutura de Decisão (SE/ENTÃO/SENÃO).
A definição de uma Variável de Controle.
